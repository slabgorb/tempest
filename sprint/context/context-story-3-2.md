## Acceptance Criteria
- advanceLevel(s) swaps geometry: s.tube = tubeForLevel(s.level) so each level actually shows its roster shape (NOT the level-1 circle).
- advanceLevel resizes the spike array to the new geometry: s.spikes = new Array(s.tube.laneCount).fill(0) — laneCount varies (12/14/15/16) across the roster, so a stale-length array is a bug.
- advanceLevel wraps the player lane into the new tube: s.player.lane = wrapLane(s.tube, s.player.lane), keeping the Claw in range when laneCount shrinks.
- Clearing a level enters 'warp' mode (warp.progress 0->1 at WARP_SPEED) instead of advancing immediately; advanceLevel runs on warp completion and resets warp.progress to 0, mode to 'playing'.
- Regression guard the 3-1 suite could not cover: a test asserts that after a level transition out.tube.laneCount === tubeForLevel(out.level).laneCount AND out.spikes.length === tubeForLevel(out.level).laneCount.
